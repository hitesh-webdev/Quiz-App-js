# Simple Quiz App

A fun and interactive quiz application built using HTML, CSS, and JavaScript.  
The app presents multiple-choice questions and provides instant feedback with a final score at the end.

## Features
- Multiple-choice questions with four options
- Dynamic question rendering using JavaScript
- Instant visual feedback for correct and incorrect answers
- Score calculation based on correct answers
- Next button to navigate between questions
- Restart quiz option after completion
- Clean and modern dark-themed UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How It Works
- Questions and answers are stored in a JavaScript array
- Each question is displayed dynamically on the page
- Users select an answer by clicking on an option
- Correct answers are highlighted in green and incorrect ones in red
- All options are disabled after selection to prevent multiple attempts
- The **Next** button loads the next question
- After the last question, the final score is displayed
- Users can restart the quiz using the **Play Again** button

- Arrays and objects
- DOM manipulation
- Event handling
- Conditional logic
- Dataset attributes
- Dynamic element creation
